# DCF Attendance Portal

A simple, secure, and modern attendance management portal developed for the **Department of DCF** to help faculty mark and manage student attendance efficiently.

## Features

* Secure faculty login authentication
* Subject-based access control
* Student list fetching through Google Apps Script API
* Attendance marking with Present / Absent toggle
* Overwrite confirmation for existing attendance
* Search by Register Number / Student Name
* Pagination for large student lists
* Select All / Unselect All options
* Modern dark UI with emerald theme
* Custom alert / confirmation modal
* Calendar date picker
* Show / Hide password option
* Mobile responsive design

## Tech Stack

* HTML
* CSS
* JavaScript
* Google Apps Script
* Google Sheets

## How it Works

Faculty logs in using assigned credentials.
The system verifies login using Google Apps Script, loads allowed subjects, fetches student data securely, and stores attendance directly in Google Sheets.

## Security

* No hardcoded usernames or passwords in frontend
* No public Google Sheet access
* Backend validation for login and subject permissions
* Secure attendance submission through Apps Script API

## Project Purpose

This portal was created to simplify daily attendance work for faculty with a clean interface and secure backend integration.
